# P6 行业现状
1. **活性污泥法污水处理的原理是什么？成本多高？效率多高？存在哪些弊端？为什么污泥会膨胀呢？那为什么还会采取这种方法来进行污水处理呢？** <br>活性污泥法是目前使用最广的一种污水处理方式，它的原理其实就是利用大量的微生物，在有氧的环境下把污水中的有机物分解成水和二氧化碳，同时也会生成一些新的污泥。<br>**弊端：**
   ①首先是**成本**问题，它的建设环节需要建曝气池、沉淀池，还要配鼓风机，初期投入就很高。而在运行过程中，曝气这一步非常耗电，占整个运行能耗的六成以上，所以整体运行成本也不低，一般每立方水的处理成本在0.3到0.8元之间。对中小型污水站来说，这个压力很大。<br>②第二个问题是**效率虽然还可以**，有机物的去除率能到90%左右，但在氨氮和总磷的处理上，需要靠改良工艺才能勉强稳定达标，尤其是总磷控制一直是短板。<br>③第三个问题是**污泥的副产和管理**。它每处理一批污水都会产生大量的剩余污泥，这些污泥本身也是污染物，如果处理不及时或者管理不好，会反过来影响出水水质。而且还有一个常见现象叫**污泥膨胀**，就是丝状菌大量繁殖之后，污泥变得不易沉淀，导致出水浑浊甚至超标。<br>现在还普遍采用这个方法的主要原因是它技术成熟、适用范围广，不管是生活污水还是工业废水都能用。但是我们正是看到这些痛点，才希望通过我们的材料和方法，在不增加能耗的前提下，实现更稳定、更高效、更低成本的净化处理。

2. **自然生物处理法的原理是什么？为什么会占地面积广？这种方法存在哪些弊端？为什么还会采取这种方法来进行污水处理呢？**<br>自然生物处理法，其实可以理解为在模拟自然生态的方式来净化污水。它不是靠机械设备，而是靠植物、土壤和微生物之间的协同作用，通过物理过滤、化学吸附和生物降解，把污水中的有机物和污染物去除掉。<br>这种方法的明显问题是**特别占地**。根本原因在于**传质效率和生物降解速度都比较低**。比如水在系统里流得慢，植物对污染物的吸收效率有限，为了达到处理效果，只能靠拉大面积来弥补。<br>这种方法的**弊端也不少**：  <br>①处理效率低，尤其是冬天，微生物活性下降明显，BOD<sub>5</sub>（五日生化需氧量）的去除效率会掉30%到50%；<br>②系统容易堵塞，像人工湿地或者渗滤系统，用两三年后，填料孔隙会被堵住，需要大修；<br>③有些系统还会产生**二次污染**，比如厌氧区域会放出甲烷、硫化氢，前者是温室气体，后者有臭味；<br>④它的**隐性成本高**，特别是土地的占用成本，有时甚至比设备本身更贵。<br>还有人用主要有三个理由：  <br>①它**能耗极低**，每吨水的电耗不到 0.05 度；<br>②**运维简单**，不需要专业人员，特别适合农村或者偏远地区；<br>③有一定**生态价值**，比如人工湿地还能美化环境，提升生物多样性。<br>所以它的使用不是因为效果好，而是因为便宜、简单、能“凑合用”。我们现在做的，就是在解决这些传统系统不能解决的效率、空间和可持续性问题。

3. **生物膜法污水处理的原理是什么？成本如何？效率如何？原理是什么？运维难在哪里？为什么还会采取这种方法来进行污水处理呢？**<br>生物膜法的原理，是在载体表面让微生物自然附着、繁殖，形成一层生物膜。这个生物膜会主动吸附并分解污水中的有机污染物，从而实现净化处理。它和活性污泥法不同，不需要频繁回流污泥，结构更稳定。<br>成本方面，主要花在载体材料上，比如塑料或陶瓷类的填料，每吨大概在2000到5000元之间，价格较为昂贵。<br>从处理效率来看，它的**脱氮效率大概在60%到80%之间**，**除磷在30%到60%**，整体表现比自然法好，稍逊于一些强化活性污泥法，但胜在运行更平稳。<br>它的运维成本主要有四块：<br>①**载体更换**，占到30%~40%，因为生物膜老化后要更换填料；<br>②**曝气能耗**，占25%~35%，这个是运行中的主要电耗；<br>③**污泥处置**，比活性污泥少，但仍占15%~25%；<br>④**清洗和维护**，一般用化学剂处理，成本约占10%~15%。<br>很多地方依旧采用的关键原因有两点：<br>①它比活性污泥法**产生的剩余污泥少30%左右**，运维压力小；<br>②它的系统**布置灵活**，特别适合中小规模或者农村分散型污水处理站，不需要太复杂的管理系统。<br>总的来说，生物膜法虽然工艺结构简单，但膜层更新不可控、易堵塞，导致其在农村分散处理和长期无人值守中存在严重运维难题。

# P11 解决方案
1. **为何会选择磷尾矿作为主要原料？你们是否评估过其他固废资源的性价比和环境价值？**<br>我们选择磷尾矿作为免烧陶粒的核心原料，是基于资源危机消解、技术适配升级、环境效益倍增的三重战略考量。经系统评估，磷尾矿在性价比与环境价值上，显著优于粉煤灰、钢渣等主流固废，是‘以废治废’的最优路径。<br>①我国磷尾矿堆存量超10亿吨，每年新增7000万吨，并且在云贵鄂磷矿带，尾矿库已逼近生态红线；<br>②磷尾矿成分完美匹配免烧陶粒的硅钙基骨架，其粒径小的特性，减少粉碎能耗；同时微量磷钾元素更赋予陶粒土壤改良功能——这是粉煤灰等固废不具备的附加值；<br>③国家《关于 “十四五” 大宗固体废弃物综合利用的指导意见》要求尾矿“吃干榨净”。地方政府对磷尾矿资源化项目给予税收减免与固废处置补贴。以磷尾矿为原料，既符合政策导向，又能显著降低原料成本；<br>④相比其他固废，磷尾矿在免烧陶粒领域具备成本归零优势，同时可解决大规模堆积导致的环境污染问题，可实现“减碳、节地、控污”三重效益。<br>据中科院地化所2024年评估，磷尾矿资源化优先级已升至国家固废清单首位。

2. **你们的原料全是磷尾矿吗？磷尾矿的重金属元素是如何处理的？除了重金属元素，还有其他有危害的元素吗？**<br>我们使用的原料基本全部为磷尾矿，仅添加少量助剂（3%浓度双氧水、水泥、石膏、NaOH）调控性能。<br>关于重金属的处理，我们采用“碱激活 + 凝胶固定”的复合机制，通过NaOH激活尾矿中活性SiO₂，与Ca²⁺生成C-S-H凝胶结构；<br>重金属通过以下三种方式被牢固固定：<br>①离子交换：如Pb²⁺、Cd²⁺可置换Ca²⁺，嵌入凝胶层间结构；<br>②表面络合：如Cr³⁺可与Si-O⁻或Al-O⁻形成共价络合键；<br>③凝胶包封：C-S-H网络能物理包覆游离态金属离子。<br>整体重金属固定率超过95%，具有长期环境稳定性。<br>其他可能存在的有害元素也已考虑：<br>①F⁻（氟离子）：可与Ca²⁺形成难溶的CaF₂沉淀；<br>②As（砷）：可被C-S-H网络物理包裹；<br>所以，从原料利用率到污染控制，我们实现了资源化+无害化同步处理，确保环保与材料功能并重。

1. **微米孔是如何构筑的？**<br>我们是通过原料磷尾矿中添加**胶凝材料**（硅铝酸钙）及**发泡剂**（过氧化氢），在造粒过程中形成陶粒骨架，发泡剂的分解产生气体，使得陶粒产生孔隙结构，而胶凝材料让陶粒产生初始强度，抑制气体排出，此时就使得发泡剂在原位产生孔隙。<br>Eg，就像我们做馒头，在其中添加酵母菌，而酵母菌产生气体就可以让馒头变得蓬松，内部是蜂窝状的孔洞结构，而造孔剂在其中分布是非常均匀的，就会在内部产生均匀的孔隙。

2. **为何会产生微纳米孔？**<br>①发泡剂产生的氧气是分子，很小，我们会严格控制发泡剂过氧化氢的量和分散程度以及养护温度（决定发泡时间）；<br>②在研发过程中我们遇到过技术难题，由于气体的逸出会使材料产生通道，此时会降低陶粒强度，并增大其孔径大小。

3. **为什么要构筑微米级别的孔结构？**<br>我们进行了孔径结构的对比实验。设置了不同平均孔径的陶粒样本，通过扫描电镜和生物膜蛋白含量测定分析了各样本在相同挂膜条件下的定殖速率和膜厚分布。结果显示，孔径在150μm左右的陶粒在初期定殖速率和最终膜厚方面均优于其他孔径段，验证了微孔结构有利于提供庇护位点和增加比表面积，促进生物膜形成。因此，并非仅基于理论推断，而是有实验数据支撑。

13. **不同温度和不同水量负荷的变化下对菌群活性有何影响？**<br>温度和负荷确实是影响微生物活性的关键因素。<br>我们测试表明，在10℃至35℃之间，菌群活性保持较好，25–30℃为最优温区。在低温下净化效率会略有下降，但膜不会脱落，具备良好的恢复能力。水量或有机负荷突然增加时，生物膜系统相比活性污泥法具有更强的缓冲和适应能力，短期内菌群可自我调节以恢复平衡，表现出很强的抗冲击性能。
